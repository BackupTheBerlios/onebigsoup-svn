Welcome to Local Names!

What is Local Names?

Local Names is an open data format for naming URLs.

It also allows you to network name spaces.

Where can I use Local Names?

Right now, you can use Local Names in:

  • the Firefox web browser
  • WordPress blogs
  • most wiki
  • IRC

We'd like to be able to use Local Names in e-mail, instant messengers, on community web sites, even in software code!

What are name spaces?

A "name space" is a collection of named URLs.

You can think of it as a "bag" of Local Names.

What are name spaces for?

If you talk about a lot of things, you eventually want to use names differently in different contexts.

For example, talking about "CGI" should link one place in the context of Python, and another place in the context of Perl.

With Local Names, you can link names to different URLs in different spaces.
