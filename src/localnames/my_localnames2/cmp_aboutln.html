Related Works

Launch Bar, Quick Silver

These are Mac OS X programs that you can use to start applications and find web pages.

They feature sophisticated matching algorithms, and access to almost everything on your computer.

Local Names is different; Local Names is a data format for describing names of links. That means names can be shared, and used in many programs.

ActiveWords

This is like Launch Bar or Quick Silver, but for Windows.

The differences are the same: Local Names is a data format for describing names of links. The links can be shared, and used in many programs, even across the Internet.

Local Names is free, too!
